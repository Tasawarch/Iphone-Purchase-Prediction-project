**Project Name: iPhone Purchase Prediction**

**Description:**
This is a full-stack Machine Learning project that predicts whether a customer is likely to purchase an iPhone based on their income and age. The project integrates a trained Python ML model with a React.js front-end and a Node.js + Flask backend.

**Key Features:**

Accepts manual input for single predictions.

Supports bulk predictions via Excel file upload and allows downloading a prediction results file.

End-to-end integration of frontend, backend, and ML model.

Built with Python (ML model training), React.js (UI), Flask + Node.js (backend APIs).

**Tech Stack:**

Frontend: React.js

Backend: Node.js, Flask

ML: Python (e.g., scikit-learn, pandas)

Data Handling: Excel file upload/download

**Use Case:**

Helps businesses identify potential iPhone customers based on demographics.
